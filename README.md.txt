# Commodity Market Analysis Dashboard

## Overview
An interactive data analytics dashboard for global commodity prices (1997–2025).
Built using Python, DuckDB, SQL window functions, and Streamlit.

## Features
- Multi-commodity price exploration
- SQL-based rolling averages (30D, 90D)
- Fast analytics using DuckDB
- Interactive Streamlit dashboard

## Tech Stack
- Python
- DuckDB
- SQL (window functions)
- Streamlit
- VS Code

## Project Structure
```text
app/        → Streamlit dashboard  
Analysis/   → DuckDB analytics database
